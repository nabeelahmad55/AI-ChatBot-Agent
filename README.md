#  AI Chatbot Agent

An AI-powered chatbot built with **Next.js** and **OpenAI’s GPT-5-mini**, featuring real-time streaming responses and a modern, responsive chat interface.

---

##  Features
-  Real-time AI conversations with smart, dynamic responses  
-  Next.js API integration for fast and efficient message streaming  
-  Automatically greets users based on current page content  
-  Clean, responsive UI built using React components  
-  Built-in error handling and connection fallback messages  

---

##  Tech Stack
- **Framework:** Next.js (React)
- **Language:** JavaScript (JSX)
- **AI Model:** OpenAI GPT-5-mini via `ai` SDK
- **Deployment:** Vercel
